# Hexagonal Architecture Workshop - WSC 2025

## Prerequisites
- Node.js
- pnpm

## Setup
Install dependencies:
```bash
pnpm install
```

## Running the Application in Development Mode
```bash
pnpm dev
```

## Running Tests
Run one-time:
```bash
pnpm test
```
Run in watch mode:
```bash
pnpm test:watch
```
Run with coverage:
```bash
pnpm test:coverage
```
